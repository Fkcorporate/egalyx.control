🚀 Prise en main rapide

1. Créer votre compte

Rendez-vous sur la page de connexion et créez votre compte. Vous recevrez un email de confirmation.

2. Configurer votre organisation

Une fois connecté, commencez par configurer votre organisation :

  • Informations de l'entreprise
  • Départements/services
  • Utilisateurs et rôles

3. Premiers pas

Suivez notre assistant de configuration qui vous guidera pour :

  • Créer votre première cartographie des risques
  • Configurer vos premiers indicateurs KRI
  • Planifier votre premier audit
Nouveau : Regardez notre tutoriel vidéo de prise en main (10 minutes)

🗺️ Cartographie des risques

Créer une cartographie

  1. Dans le menu, sélectionnez "Risques" > "Cartographie"
  2. Cliquez sur "Nouvelle cartographie"
  3. Donnez un titre et une description
  4. Sélectionnez la période concernée
  5. Ajoutez vos risques (manuellement ou via import Excel)

Évaluer les risques

Pour chaque risque, évaluez :

  • Probabilité : de 1 (très faible) à 5 (très élevée)
  • Impact : de 1 (mineur) à 5 (critique)
  • Détectabilité : de 1 (facile à détecter) à 5 (difficile à détecter)
  • Criticité : calculée automatiquement (Probabilité × Impact)

Visualiser la cartographie

La cartographie génère automatiquement :

  • Une matrice des risques (heat map)
  • Un tableau de Bordeaux
  • Des graphiques d'analyse

📊 Indicateurs KRI (Key Risk Indicators)

Créer un indicateur

  1. Menu "Risques" > "Indicateurs KRI"
  2. Cliquez sur "Nouvel indicateur"
  3. Définissez : nom, description, unité, fréquence de mesure
  4. Configurez les seuils d'alerte (vert, orange, rouge)
  5. Associez l'indicateur à un ou plusieurs risques

Saisir des valeurs

Vous pouvez saisir les valeurs :

  • Manuellement via l'interface
  • Via import Excel/CSV
  • Automatiquement via API

Tableaux de bord

Les indicateurs sont visualisés sur des tableaux de bord dynamiques avec :

  • Évolution dans le temps
  • Alertes visuelles (couleurs)
  • Seuils personnalisables

✅ Gestion des audits

Planifier un audit

  1. Menu "Audits" > "Planification"
  2. Cliquez sur "Nouvel audit"
  3. Renseignez : titre, dates, équipe d'audit
  4. Définissez le périmètre (processus, risques concernés)
  5. Créez votre checklist de contrôle

Réaliser un audit

Pendant l'audit :

  • Remplissez les fiches de contrôle
  • Ajoutez des constatations
  • Téléchargez des pièces justificatives
  • Générez des recommandations

Plans d'action

Pour chaque recommandation, créez un plan d'action :

  • Responsable désigné
  • Échéance
  • Étapes à suivre
  • Suivi de l'avancement

🤖 Intelligence Artificielle

Analyse prédictive

Notre IA analyse vos données historiques pour :

  • Prédire l'évolution des risques
  • Détecter des anomalies
  • Identifier des corrélations cachées

Recommandations automatiques

L'IA génère des recommandations :

  • Basées sur les meilleures pratiques
  • Adaptées à votre secteur
  • Priorisées selon l'urgence

Activer l'IA

L'IA est disponible dans les formules Premium et Enterprise. Pour l'activer :

  1. Menu "Paramètres" > "IA & Analytics"
  2. Activez les modules souhaités
  3. Laissez l'IA analyser vos données (24-48h)
