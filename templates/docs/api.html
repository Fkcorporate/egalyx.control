Introduction

L'API REST d'Egalyx Control vous permet d'intégrer nos fonctionnalités d'audit et de gestion des risques à vos propres applications et processus.

URL de base : https://api.egalyx-control.com/v1

Formats supportés

L'API accepte et retourne des données au format JSON uniquement.

Authentification

L'API utilise des clés d'API pour l'authentification. Chaque requête doit inclure votre clé dans l'en-tête HTTP.

Authorization: Bearer VOTRE_CLE_API

Obtenir une clé API

  1. Connectez-vous à votre espace client
  2. Allez dans Paramètres > Intégrations > API
  3. Cliquez sur "Générer une nouvelle clé"
  4. Copiez votre clé (elle ne sera plus affichée)
Important : Ne partagez jamais votre clé API. En cas de compromission, regénérez-la immédiatement.

Endpoints principaux

Risques

Méthode Endpoint Description
GET /risques Liste tous les risques
GET /risques/{id} Détail d'un risque
POST /risques Créer un nouveau risque
PUT /risques/{id} Mettre à jour un risque
DELETE /risques/{id} Supprimer un risque

Audits

Méthode Endpoint Description
GET /audits Liste tous les audits
GET /audits/{id} Détail d'un audit
GET /audits/{id}/statistiques Statistiques d'un audit
POST /audits Créer un audit

KRI (Indicateurs)

Méthode Endpoint Description
GET /kri Liste des indicateurs
GET /kri/{id}/valeurs Valeurs d'un indicateur
POST /kri/{id}/valeurs Ajouter une valeur

Exemples

Python

import requests

API_KEY = "votre_cle_api"
BASE_URL = "https://api.egalyx-control.com/v1"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

# Récupérer tous les risques
response = requests.get(f"{BASE_URL}/risques", headers=headers)
risques = response.json()

# Créer un nouveau risque
nouveau_risque = {
    "reference": "RISK-2024-001",
    "intitule": "Risque de non-conformité RGPD",
    "categorie": "Conformité",
    "probabilite": 3,
    "impact": 4
}

response = requests.post(f"{BASE_URL}/risques", 
                         json=nouveau_risque, 
                         headers=headers)

JavaScript

const API_KEY = 'votre_cle_api';
const BASE_URL = 'https://api.egalyx-control.com/v1';

const headers = {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
};

// Récupérer tous les audits
fetch(`${BASE_URL}/audits`, { headers })
    .then(response => response.json())
    .then(audits => console.log(audits));

// Créer un audit
const nouvelAudit = {
    reference: 'AUDIT-2024-001',
    titre: 'Audit conformité RGPD',
    date_debut: '2024-04-01',
    date_fin: '2024-04-15'
};

fetch(`${BASE_URL}/audits`, {
    method: 'POST',
    headers: headers,
    body: JSON.stringify(nouvelAudit)
});
